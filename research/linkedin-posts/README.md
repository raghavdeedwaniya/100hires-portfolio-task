# LinkedIn posts (manual collection)

LinkedIn blocks automated scraping. Everything in this folder is collected **manually** by opening each expert's profile, filtering to their posts, and pasting the strongest on-topic content into the per-author markdown file below.

## Format per file
```
# <Author Name>
- **LinkedIn:** <url>
- **Date collected:** YYYY-MM-DD

---

## Post 1
- **URL:** <post permalink>
- **Date:** YYYY-MM-DD
- **Why it's in the corpus:** <one line>

<paste full post text here>

---
```

## Collection targets
Quality over quantity: 5–10 posts per author from the last 3–6 months, on cold outreach / cold email / cold calling / outbound systems.
